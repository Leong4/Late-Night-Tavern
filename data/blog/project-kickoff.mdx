## 为什么做这个项目？

LLM 领域日新月异，各种框架、插件、数据源不断涌现。我希望能构建一套系统，**自动追踪技术趋势，分析结构化演化路径**，从而：

- 跟踪 RAG、Agent 等核心技术的微演化
- 发掘不被大模型覆盖的中长尾趋势
- 构建面向个人开发者的“技术雷达”工具链

## 项目目标

构建一个“轻量级、可自动更新、可视化输出”的 LLM 技术趋势追踪系统，具备以下能力：

- 自动抓取 arXiv / GitHub / Twitter / Reddit / 知乎 等数据源
- 支持标签聚类、关键词热度趋势图
- 支持嵌入式相似度召回 + GPT 摘要
- 面向非专家用户也能读懂的趋势可视化

## 技术选型 & 架构初稿

- Next.js + Tailwind 作为前端展示框架
- Node + Python 混合任务爬虫
- SQLite + 向量数据库（可能是 Qdrant）作为本地存储方案
- 本地 LLM（DeepSeek/Qwen）进行摘要与重排

## 下一步计划

1. 梳理关键数据源（含过滤规则）
2. 完善技术标签体系
3. 构建每日同步更新机制
4. 将雷达结果可视化，并展示在站内
